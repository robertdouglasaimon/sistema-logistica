Comando para abrir a porta ONLINE para mostrar o projeto com NGROK.
Depois que fizer a instalação do NGROK via npm install ngrok e...
Depois de ter baixado o ngrok da internet e ter logado com o token, criar um arquivo chamado
"ngrok.yml" e colocar os seguintes paramentros:

version: '2'
agent:
  authtoken: 31Knlg5W3wgjaOJwQNSLImgyQxA_38LTuKRfuh6rC4FfEgMtX
tunnel:
  next: (Aqui coloca a tecnologia que você ta usando, no meu caso como usei Next.. Foi Next!)
    addr: 3000  (Aporta do localhost que vai ficar online)
    protocol: http  (E o tipo de protocolo que vai executar)

E por fim, no terminal do vscode ou do power shell, chamamos o comando abaixo junto da porta do projeto para gerar o link do servidor temporario com o link online do local host:

" .\ngrok http 3000 "

E é isso, seguindo esses passos um servidor online do link local será criado para mostrar o projeto para alguém sem precisar subir pro github.
